1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
   Answer: getElementById(), getElementsByClassName(), querySelector(), and querySelectorAll() are JavaScript methods used to select HTML elements from the DOM. The getElementById() method selects a single element using a specific id and returns only one element. It is very fast because an id is usually unique within a webpage. On the other hand, getElementsByClassName() selects all elements that share a specific class name and returns them as an HTMLCollection, which is a live collection. This means if the DOM changes, the collection updates automatically.

The querySelector() method uses CSS selectors to select the first matching element. It is more flexible because it allows you to use id, class, tag, attribute, and other CSS selector patterns. Meanwhile, querySelectorAll() also uses CSS selectors but returns all matching elements as a NodeList. Unlike HTMLCollection, this NodeList is static, meaning it does not automatically update if the DOM changes later. In summary, use getElementById() to select a single element by id, getElementsByClassName() to select multiple elements by class, and querySelector() or querySelectorAll() when you need more flexible CSS-based selection.
